# Financial Dashboard (Excel)

## 🔍 Overview

This project is an interactive financial dashboard built using Microsoft Excel. It helps in analyzing financial data through dynamic KPIs, pivot tables, and slicers, making insights easy to understand and visualize.

## Features

* Interactive slicer-based filtering
* Dynamic KPIs linked with pivot tables
* Clean and professional dashboard layout
* Real-time data updates
* Error-free and stable data connections

## Tools Used

* Microsoft Excel
* Pivot Tables
* Slicers
* GETPIVOTDATA

## Purpose

The aim of this project is to demonstrate skills in Excel-based data analysis, dashboard creation, and financial reporting. It showcases the ability to convert raw data into meaningful insights for decision-making.
